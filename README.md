🍳 PixelFoodie AI - Smart Recipe Predictor

Welcome to PixelFoodie AI. It is a machine learning-powered web application that predicts the best recipes based on your dietary preferences, meal timings, and cuisine interests!

🌟 Overview
I built this project to demonstrate how machine learning models can be seamlessly integrated into a web-based user interface. 
PixelFoodie AI helps users decide what to eat by analyzing their choices (Veg/Non-Veg, Breakfast/Lunch/Snacks, and Cuisine type).

🚀 Live Demo
Check out the live project here: [PixelFoodie AI Predictor](https://pixelfoodie-ai-predictor.onrender.com)

✨ Features
Intelligent Prediction: Uses a python ML model to suggest dishes.
Modern UI: Clean, responsive design with a glassmorphism effect.
Cuisine Diversity: Supports South Indian, North Indian, and Western categories.
Fast Response: Optimized backend for quick recipe suggestions.

🛠️ Tech Stack
Frontend: HTML5, CSS3, JavaScript
Backend: Python (Flask) 
Machine Learning: Scikit-learn, Pandas, NumPy
Deployment: Render.com
